Estrutura basica do HTML

(Voltar)

Html tem uma estrura base que é definida por as seguintes tags:

  1. Tag < html > Para o navegador entender que tipo de arquivo é < / html >
  2. Tag < head > ou seja o cabeçalho da nossa pagina, não visivel para o cliente < / head >
  3. Tag < body > É o corpo, tudo o que estiver dentro do corpo é visivel para o cliente < / body >

Listas ordenada e não ordenadas

(Voltar)

As listas podem nos ajudar a organizar melhor as informações da nossa pagina para definir uma lista usamos a tag < li > < / li >

A tag < ul > Indica que nossa lista ( li ) é não ordenada ou seja não ordena por numeros o nosso texto < / ul >

A tag < ol > Indica que nossa lista ( li ) é ordenada quer dizer que nossa lista vai ser odenada por numeros do (1) em diante < / ol >
